# Energy Bill Calculator
A simple Python calculator that estimates energy bills based on usage and rates.
Built as part of my Python learning journey toward Energy Market Analysis and AI model development

## What it does
- Takes daily energy use (kWh) as input
- Takes electricity rates (cents per kWh) as input
- Takes number of days in billing period as input
- Calculates total energy used and total bill amount
